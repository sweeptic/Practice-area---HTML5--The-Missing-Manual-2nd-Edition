The flex-grow property is used to allocate leftover space once the widths of the flex items are totaled (see the box on page 549).

You can think of the flex-basis value as a kind of minimum width for a flex item.

If you don’t set the flex property on flex items within a flex container, browsers provide a default setting, which is the equivalent of: flex: 0 1 auto; With this setting, the width of each flex item is automatically determined by the content inside it.
